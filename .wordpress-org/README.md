# WordPress.org Plugin Directory Assets

This directory contains images displayed on the [WordPress.org plugin page](https://wordpress.org/plugins/chip-for-woocommerce/).

## File naming conventions

- `banner-772x250.png`   — Plugin page header banner (normal resolution)
- `banner-1544x500.png`  — Plugin page header banner (high-DPI / retina)
- `icon-128-128.png`     — Plugin icon (normal resolution)
- `icon-256-256.png`     — Plugin icon (high-DPI / retina)
- `screenshot-1.png`     — Screenshot #1 (referenced in readme.txt)
- `screenshot-2.png`     — Screenshot #2
- ...and so on

> **Do not delete or rename files here** unless you also update the
> `== Screenshots ==` section in `readme.txt`.

These files are synced to `https://plugins.svn.wordpress.org/chip-for-woocommerce/assets/`
by the deploy workflow (`.github/workflows/deploy.yml`).
